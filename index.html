How to play:
It's a counter game where the object is to find pairs.
When the game begins, all pictures are hidden.
To play:
1. Select two cards to try to match the pictures.
2. If you match the pictures yoy can go again.
3. If cards don't match they turn back.
4. Wnen you finds all pairs, you win.
5. Don't forget to have fun!
